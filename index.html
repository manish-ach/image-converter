🖼️ PHP Image Converter

High-Performance Image Processing via Rust FFI

🦀 Rust 🐘 PHP ⚡ FFI 📦 PHAR 🔄 Multi-Format

🎯 Project Overview

A revolutionary image processing solution that combines the performance of Rust with the accessibility of PHP. This project demonstrates advanced Foreign Function Interface (FFI) integration, creating a portable, high-performance image converter that supports multiple formats and provides terminal-based interactive features.

⚡ Key Features

🚀 Blazing Fast Performance

Rust-powered image processing delivers 6-10x faster conversion compared to pure PHP implementations.

🔄 Multi-Format Support

Seamlessly convert between JPG, PNG, and WEBP formats with optimized compression algorithms.

📦 Portable Binary

Single PHAR executable with embedded dependencies - no installation required, runs anywhere.

🖥️ Terminal Integration

Interactive file selection with fzf and native image preview in Kitty terminal.

💻 Code Example - FFI Integration

// PHP FFI Integration
$ffi = FFI::cdef("
int convert_image(const char* input_path,
const char* output_path,
const char* format);
", __DIR__ . "/lib/librust_image_converter.so");

// Call Rust function from PHP
$result = $ffi->convert_image($input, $output, $format);

if ($result === 0) {
echo "✅ Conversion successful!\n";
} else {
echo "❌ Conversion failed!\n";
}

🛠️ Installation & Usage

1
Clone Repository:
git clone https://github.com/manish-ach/image-converter.git
2
Build Everything:
make (Compiles Rust library + Creates PHAR binary)
3
Run Converter:
./image-converter (Interactive mode with fzf selection)
4
Global Install:
sudo cp image-converter /usr/local/bin/

🔧 Technical Innovations

🔗 FFI Bridge Design

Custom C-compatible interface exposing Rust functions to PHP with proper memory management and error handling.

📦 PHAR Packaging

Everything bundled into a single executable: PHP code, Rust library, and dependencies - true portability.

🎯 Cross-Platform Build

Automated Makefile system detects OS and compiles appropriate library extensions (.so, .dll, .dylib).

📺 Terminal Enhancement

Kitty terminal integration provides in-terminal image previews using the Kitty graphics protocol.
